# CRM Backend

A production-style Customer Relationship Management backend built with Java and Spring Boot.

## Overview

This project is a backend CRM system designed to manage customers, deals, follow-ups, interactions, dashboards, and audit logs.

The goal of this project is to practice building a real-world backend system with authentication, authorization, business rules, testing, Docker, and CI/CD.

## Tech Stack

- Java 17+
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA / Hibernate
- MySQL
- Maven
- Docker
- Redis / Caching
- JUnit 5
- Mockito
- Testcontainers
- GitHub Actions
- Swagger / OpenAPI

## Main Features

- Authentication and authorization
- Role-based access control
- Row-level ownership checks
- Customer management
- Deal pipeline management
- Interactions and follow-ups
- Search, filtering, and pagination
- Dashboard and analytics
- Audit logging
- Dockerized environment
- CI pipeline

## Architecture

This project follows a modular monolith approach with layered architecture.

Main layers:

- Controller layer
- Service layer
- Repository layer
- Entity layer
- DTO layer
- Mapper layer
- Security layer
- Exception handling layer

For more details, see: `docs/architecture.md`
